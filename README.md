# archive-project

한국어로 배우는 쉬운 컴퓨터공학

이 웹사이트는 다음 기능을 제공합니다.
1. 컴퓨터 공학적 지식을 쉽게 찾을 수 있습니다.
2. 여러 유저들이 쓴 정보들을 시리즈에 맞게 볼 수 있습니다.
3. 글을 쓰고 공유하고 피드백받을 수 있습니다.
4. 현 상태에 가장 알맞는 글을 추천해드립니다.

## 프로젝트의 의의

이 프로젝트(웹사이트)는 다음 목적을 위해 작성되었습니다.
1. 프론트엔드적 지식의 발전
2. 프로젝트에서 해야하는 기본적인 행위
3. 백엔드과의 연결 구상 및 실현

## 빠른 시작
프로젝트를 로컬에서 빠르게 실행하려면 아래 절차를 따르세요.

## 설치
1. 레포지토리를 클론합니다.
```bash
git clone https://github.com/soeld0920/archive-project.git
cd archive-project
```
2. 의존성을 설치합니다.
```bash
npm install
# 또는
yarn install
```

## 로컬 실행
개발 서버를 실행합니다.
```bash
npm run dev
# 또는
yarn dev
```
배포용 빌드는:
```bash
npm run build
# 또는
yarn build
```

## 프로젝트 구조
- src/ — 애플리케이션 소스
- public/ — 정적 파일
- README.md — 프로젝트 설명
- package.json — 스크립트 및 의존성

## 기술 스택
- 프론트엔드: React
- 빌드 도구: Vite
