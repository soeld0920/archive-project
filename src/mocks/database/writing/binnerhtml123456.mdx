## 간단한 HTML 구조와 태그: 초보자 안내

### 목표
- HTML 기본 문서 구조 이해
- 자주 쓰는 태그 설명 (div, p, a, img, ul/li, h2 등)

### 기본 문서 구조
아래는 최소한의 HTML 문서 구조 예시입니다:

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="utf-8" />
    <title>문서 제목</title>
  </head>
  <body>
    <h2>페이지 제목</h2>
    <p>간단한 단락입니다.</p>
  </body>
</html>
```

### 자주 쓰는 태그
- 제목: `h1`~`h6` (이 글은 H2로 시작했으므로 큰 제목은 사용하지 않음)
- 단락: `p`
- 링크: `a href="URL"`
- 이미지: `img src="경로" alt="설명"`
- 목록: `ul`(순서 없는 목록), `ol`(순서 있는 목록), `li`(목록 항목)

### 실습: 간단한 프로필 카드 만들기
```html
<div class="card">
  <img src="/img/user.png" alt="사용자 이미지" />
  <h2>이름</h2>
  <p>간단한 소개 문장.</p>
  <a href="#">더 보기</a>
</div>
```

### 팁
- 시멘틱 태그(예: `header`, `main`, `footer`)를 사용하면 접근성과 구조 이해에 도움이 됩니다.
- CSS로 스타일을 꾸미기 전에는 HTML 구조를 먼저 잘 설계하세요.
