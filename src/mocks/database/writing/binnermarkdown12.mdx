## Markdown 기초: 초보자용 빠른 입문

### 이 글에서 배울 것
- Markdown의 기본 문법
- 코드 블록, 링크, 이미지 삽입 방법
- 실습 예제

### 왜 Markdown을 배우나요
Markdown은 간단한 문법으로 문서(README, 블로그 등)를 작성할 수 있게 해 줍니다. 가독성이 좋고, 많은 플랫폼이 Markdown을 지원합니다.

### 기본 문법
- 제목: `#` 대신 이 글은 H2(`##`)로 시작합니다.
- 굵게: `**굵게**`
- 기울임: `*기울임*`
- 목록: `- 항목`

#### 코드 블록
```javascript
function hello() {
  console.log('안녕하세요');
}
```

#### 링크와 이미지
- 링크: `[링크텍스트](https://example.com)`
- 이미지: `![대체텍스트](/path/to/image.png)`

### 간단 연습
다음 내용을 `README.md`에 적어 보세요:

```
## 프로젝트 제목
간단한 설명 한 줄.

### 사용법
- 설치 방법
- 실행 방법
```

### 팁
- GitHub에서 미리보기를 보며 작성하면 실수 줄이기 좋습니다.
- 긴 문서는 섹션을 나눠서 H2/H3을 사용하세요.
