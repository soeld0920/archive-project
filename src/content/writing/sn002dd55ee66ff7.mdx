## 핵심 포인트

\n

\n          
- 인접 구조 선택

        
      

      
        
## 예제 코드

        <CodeBlock>{`function bfs(start, graph) {
  const q = [start];
  const dist = { [start]: 0 };
  while (q.length) {
    const cur = q.shift();
    for (const nxt of graph[cur] || []) {
      if (dist[nxt] == null) {
        dist[nxt] = dist[cur] + 1;
        q.push(nxt);
      }
    }
  }
  return dist;
}`}</CodeBlock>
